ReAct runtime 链路稳定性测试集：Case 4-10

说明：
1. 本文件仿照 case1-3，覆盖项目主要 intent / tool-chain / stop condition。
2. 当前真实 graph 主链：
   metrics_init -> router -> react_init -> ...ReAct loop... -> context_builder -> generator -> metrics_finalize -> memory_writer
3. ReAct loop 标准形态：
   reason -> action_selector -> tool_executor -> observation_compressor -> react_control
   若第二轮 reason 判断信息足够，则：
   reason -> action_selector -> react_control
4. 所有 case 都建议用 checkpoint history 断言节点全链路，用 tool_results / observations / runtime_metrics 断言工具链。
