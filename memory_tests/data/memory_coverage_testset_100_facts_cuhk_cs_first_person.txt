记忆系统测试集（第一人称用户输入版）
说明：
1. 本测试集用于评测 agent memory 的写入覆盖率、可检索覆盖率、更新覆盖率与选择性遗忘能力。
2. 角色设定：用户是一名想申请香港中文大学（CUHK）的本科计算机相关学生。
3. 本文件共 100 条测试事实，其中：
   - 1-25：静态用户画像（应记）
   - 26-50：多轮任务状态（应记）
   - 51-75：跨 session 事实（应记）
   - 76-90：会变化的事实（应记，含更新关系）
   - 91-100：噪音对照样本（不应记）
4. 每条均包含：ID、类别、优先级、是否应记、用户输入表述。
5. “不应记”样本用于测试 selective forgetting / noise control，不计入应记覆盖率分母。

附：建议评测方式
==================================================

1. 写入覆盖率：
   统计 1-90 中应记事实，有多少被正确写入 memory。

2. 可检索覆盖率：
   针对 1-90 中每条事实设计 query，测试是否可在 top-k 中召回。

3. 更新覆盖率：
   针对 76-90，测试系统是否优先保留并回答最新版本事实。

4. 噪音控制率：
   测试 91-100 是否被错误写入长期记忆。

5. 最终建议：
   - 1-90 作为 gold memory facts
   - 91-100 作为 negative / noise samples
