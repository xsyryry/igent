记忆系统可检索覆盖率测试查询集（50条，含标准答案）
说明：
1. 本查询集用于测试 memory retrieval 的 top-k 召回能力。
2. 每条 query 后附标准答案，可用于人工核对或自动评测。
3. 查询对象对应上一份 1-90 的应记事实集合。
4. 查询采用自然语言表达，尽量模拟真实用户会对 agent 提出的问法。

==================================================
一、静态用户画像相关查询（1-15）
==================================================

001. Query：你还记得我最想申请的学校是哪一所吗？
标准答案：香港中文大学（CUHK）

002. Query：我本科读的是什么方向来着？
标准答案：计算机相关专业

003. Query：你记得我以后想做什么类型的工作吗？
标准答案：技术研发或算法工程相关工作

004. Query：你之前记住我申请时优先考虑哪类硕士项目了吗？
标准答案：计算机科学相关硕士项目

005. Query：除了计算机科学，我还会考虑哪些相近项目？
标准答案：人工智能、数据科学、软件工程等相近项目

006. Query：你还记得我平时更偏好用中文还是英文交流吗？
标准答案：更偏好中文交流，但接受英文材料训练

007. Query：我之前说过我更喜欢什么风格的建议，你还记得吗？
标准答案：清晰、具体、可执行、分步骤的建议，不喜欢空泛建议

008. Query：你记得我是不是能接受强度比较高的每日学习计划？
标准答案：能接受

009. Query：我准备考的是雅思还是托福？
标准答案：雅思

010. Query：你还记得我的雅思总分最低目标是多少吗？
标准答案：6.5

011. Query：我的理想雅思目标分是多少来着？
标准答案：7.0

012. Query：我对雅思单项最低分有什么要求？
标准答案：单项尽量不要低于 6.0

013. Query：你记得我最担心雅思哪两个单项拖后腿吗？
标准答案：口语和写作

014. Query：我为什么更倾向高性价比备考方案，你还记得吗？
标准答案：因为家庭经济条件中等，预算需要控制

015. Query：你记得我会不会优先使用免费或低成本资源？
标准答案：会优先使用免费或低成本资源

二、多轮任务状态相关查询（16-28）
==================================================

016. Query：你还记得我当前阶段最核心的任务是什么吗？
标准答案：制定港中文申请前的雅思备考计划

017. Query：这轮里我已经决定先考雅思还是先考托福？
标准答案：先考雅思

018. Query：我这阶段是想先补口语还是先补写作？
标准答案：先补口语，再同步推进写作

019. Query：你记得我现在采用的备考策略是什么吗？
标准答案：弱项优先、四项并行

020. Query：我现在是不是希望每天都有可执行的小任务？
标准答案：是

021. Query：你记得我现在想先用什么类型的资源来练习吗？
标准答案：题库和公开资料等低成本资源

022. Query：我现在的口语训练要求是只练 Part 2，还是 Part 1 到 Part 3 都要覆盖？
标准答案：Part 1、Part 2、Part 3 都要覆盖

023. Query：你还记得我写作训练是怎么拆分的吗？
标准答案：按 Task 1 和 Task 2 分开安排

024. Query：我现在希望你从哪里抽取剑雅真题给我练？
标准答案：从本地题库中抽取

025. Query：现在写作真题是在线临时抓，还是已经改成独立命令存到本地库了？
标准答案：已经改成独立命令爬取并存到本地库

026. Query：我现在导出题目时，是优先从本地 JSON 读取，还是再去联网抓网页？
标准答案：优先从本地 JSON 读取

027. Query：你记得我当前对 PDF 导出格式有什么两个关键要求吗？
标准答案：不同题目之间要有明显间隔或分页；每道题都要有明确序号

028. Query：我现在的学习进展是按天评估还是按周评估？
标准答案：按周评估

==================================================
附：建议使用方式
==================================================

1. 对 50 条 query 分别执行 memory retrieval。
2. 检查 gold fact 是否出现在 top-k 结果中。
3. 统计：
   - Hit@1
   - Hit@3
   - Recall@5
   - MRR
4. 对更新类查询（41-50），额外检查是否召回的是最新版本事实，而不是旧版本事实。
5. 若做问答评测，可将“标准答案”作为 reference answer 做人工或自动核对。
