写作抽题链测试集：Case 1-3（按当前真实 ReAct runtime 修正版）

说明：
1. 本文件只覆盖“雅思大作文抽题”成功链路的 case 1-3。
2. 当前真实 graph 链路：
   metrics_init -> router -> react_init -> reason -> action_selector -> tool_executor -> observation_compressor -> react_control -> context_builder -> generator -> metrics_finalize -> memory_writer
3. 对 writing_practice，当前 react_max_steps = 3。
4. 抽题工具当前只支持 writing.get_random_task2_prompt(args.essay_type)，不支持 topic_filter / task_type。
5. 成功抽题后，memory_writer 从 tool_results.get_random_task2_prompt.topic.id 写入 active_writing_topic_id。

==================================================
CASE 1：基础抽题成功链
==================================================

case_id: writing_prompt_001_real
category: normal_task_chain
title: 给我一道雅思大作文题

user_inputs:
1. 给我一道雅思大作文题

pre_state:
- study_context.active_writing_topic_id = null
- writing_review_state = {}
- retrieved_docs = []
- tool_results = {}
- observations = []

mock_setup:
- tool: writing.get_random_task2_prompt
- args:
  - essay_type = null
- returns:
  success: true
  topic:
    id: task2_001
    essay_type: 观点类
    exam_date: unknown
    prompt_text: Some people think that universities should only offer subjects that are useful for employment. To what extent do you agree or disagree?
  requested_essay_type: null
  message: 已为你随机抽取一道大作文题。

expected_route:
- metrics_init 初始化 runtime_metrics.started_at_ms
- router.intent = writing_practice
- react_init.react_active = true
- react_init.react_max_steps = 3
- reason 第 1 步判断需要抽题
- action_selector.selected_tool_call = writing.get_random_task2_prompt
- tool_executor 执行成功，tool_results.get_random_task2_prompt.success = true
- observation_compressor 生成 last_observation.success = true
- react_control 继续进入下一轮 reason
- reason 第 2 步判断信息足够
- action_selector 不再选择工具
- react_control.finish_reason = enough_information
- context_builder 写 answer_context
- generator 返回题目
- metrics_finalize 写 runtime_metrics.current
- memory_writer 写 study_context.active_writing_topic_id = task2_001

required_tools:
- writing.get_random_task2_prompt

forbidden_tools:
- rag.retrieve_knowledge
- web_search.search_web
- data.collect_data
- db.get_mistake_records

expected_state_updates:
- tool_results.get_random_task2_prompt.success = true
- last_observation.success = true
- observations.length = 1
- react_finish_reason = enough_information
- answer_context 存在
- context_summary 不存在
- final_answer 包含 prompt_text
- study_context.active_writing_topic_id = task2_001

expected_metrics:
- runtime_metrics.current.task_success = true
- runtime_metrics.current.counts.steps = 1
- runtime_metrics.current.counts.selected_tool_calls = 1
- runtime_metrics.current.duplicate_tool_call_rate = 0
- runtime_metrics.current.fallback_success_rate = 0

pass_criteria:
- final_success = true
- required_tools 全命中
- forbidden_tools 全未命中
- observations.length = 1
- react_step <= 3
- fallback_used = false
- retry_count = 0

CASE 3：已有 active topic 时抽新题
==================================================

case_id: writing_prompt_003_real
category: state_transition
title: 已有 active topic 时，用户要求再来一道新题

policy_assumption:
- 当前实现会在抽题成功后用新 topic 覆盖 study_context.active_writing_topic_id。
- 当前实现不会额外记录 previous_topic_id。

user_inputs:
1. 再给我一道新的雅思大作文题

pre_state:
- study_context.active_writing_topic_id = task2_old_001
- study_context.active_writing_prompt = An increasing number of people choose to work from home...
- writing_review_state = {}

mock_setup:
- tool: writing.get_random_task2_prompt
- args:
  - essay_type = null
- returns:
  success: true
  topic:
    id: task2_003
    essay_type: 双边讨论
    exam_date: unknown
    prompt_text: Some people think that children should start school as early as possible, while others believe they should start later. Discuss both views and give your own opinion.
  requested_essay_type: null
  message: 已为你随机抽取一道大作文题。

expected_route:
- metrics_init
- router.intent = writing_practice
- react_init.react_active = true
- reason 第 1 步判断为重新抽题，不是作文提交

- tool_executor 成功
- observation_compressor.last_observation.success = true
- react_control 继续
- reason 第 2 步判断信息足够
- react_control.finish_reason = enough_information
- context_builder
- generator
- metrics_finalize
- memory_writer 用 task2_003 覆盖旧 active_writing_topic_id

required_tools:
- writing.get_random_task2_prompt

forbidden_tools:
- writing.prepare_task2_review_context
- writing.review_task2_submission

expected_final_answer_constraints:
- 明确给出新题
